# 重构任务规划

## 概述

本项目将 EPUB 翻译系统从现有的占位符架构重构为**树状递归分块**架构。

**核心变更**：
- 分块：从占位符 `[idN]` 改为树状递归分块
- 位置记录：xpath 用于记录文本位置（验证用）
- 持久化：从 chunk 级保存改为文档级 Manifest

---

## 实施计划

### Phase 1：基础设施（可并行 3 个 subagent）

#### 任务 1.1：`engine/item/tree.py`

**功能**：
- `TreeNode` dataclass：tag, attributes, children, parent, text
- `parse_html(html: str) -> TreeNode`：使用 html.parser 解析
- `to_html() -> str`：序列化为 HTML
- `decode_entities()` / `encode_entities()`
- `is_text_node()` / `is_element_node()` / `is_self_closing()`
- `_collect_text_nodes() -> list[TreeNode]`
- `find_by_xpath(xpath: str) -> Optional[TreeNode]`：按绝对路径查找节点

**验证**：
```bash
python -c "from engine.item.tree import parse_html, TreeNode; print('OK')"
ruff check engine/item/tree.py
```

---

#### 任务 1.2：`engine/item/xpath.py`

**功能**：
- `XPATH_PATTERN` 正则
- `is_valid_xpath(xpath: str) -> bool`
- `parse_xpath(xpath: str) -> tuple[str, str, int]` 解析为 (路径前缀, 类型, 索引)
  - 类型为 "text" 或 "attr"
  - 索引为数字

**验证**：
```bash
python -c "from engine.item.xpath import is_valid_xpath; print(is_valid_xpath('html/body/div[0]:text:0'))"
ruff check engine/item/xpath.py
```

---

#### 任务 1.3：`engine/item/token.py`

**功能**：
- `MAX_TOKEN_LIMIT = 1000`
- `SAFE_LIMIT = int(MAX_TOKEN_LIMIT * 0.85)` = 850
- `estimate_tokens(text: str) -> int` 使用 tiktoken
- `split_long_text(text: str, max_tokens: int) -> list[str]` 按单词边界拆分

**验证**：
```bash
python -c "from engine.item.token import estimate_tokens; print(estimate_tokens('Hello'))"
ruff check engine/item/token.py
```

---

### Phase 2：核心分块

#### 任务 2.1：`engine/item/chunker.py`

**功能**：
- `chunk_html(html: str, token_limit: int = SAFE_LIMIT) -> list[ChunkState]`
- `chunk_tree(root: TreeNode, token_limit: int) -> list[ChunkState]`
- `add_context_to_chunks(chunks: list[ChunkState]) -> list[ChunkState]`

**核心算法**：递归遍历 + 贪心合并

**验证**：
```bash
python -c "from engine.item.chunker import chunk_html; print(len(chunk_html('<p>test</p>')))"
pytest tests/engine/item/test_chunker.py -v
ruff check engine/item/chunker.py
```

---

#### 任务 2.2：`engine/item/nav.py`

**功能**：
- `NavParser` class
- `is_nav_file(html: str) -> bool`
- `extract_nav_points(html: str) -> list[dict]`
- `preserve_content_attrs(html: str) -> dict[int, str]`
- `rebuild_nav(html: str, translations: dict[int, str]) -> str`
  - `translations`: {文本节点索引: 翻译文本}

**验证**：
