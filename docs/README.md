# EPUBox Documentation

EPUBox is a sophisticated EPUB translation system with advanced HTML processing and translation capabilities.

## Documentation Structure

- `architecture/` - System architecture and design documents
- `models/` - Data model designs
- `features/` - Core feature documentation
- `interfaces/` - API interface documentation
- `testing/` - Testing related documentation
- `observability/` - Logging and monitoring design
- `development/` - Development guidelines and setup

## Getting Started

1. Check `development/setup.md` for environment setup
2. Review `architecture/overview.md` for system overview
3. Explore `features/` for specific component details
