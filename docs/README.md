# EPUBox Documentation

This directory contains the comprehensive documentation for the EPUBox project.

## Directory Structure

```
docs/
├── architecture/           # 系统架构相关文档
│   ├── overview.md        # 系统概述
│   ├── components.md      # 组件设计
│   └── data-flow.md       # 数据流设计
│
├── api/                   # API相关文档
│   ├── endpoints.md       # API端点设计
│   └── schemas.md         # 数据模型定义
│
├── translation/           # 翻译服务相关文档
│   ├── providers.md       # 翻译服务提供商集成
│   ├── strategies.md      # 翻译策略
│   └── error-handling.md  # 错误处理机制
│
├── deployment/            # 部署相关文档
│   ├── setup.md          # 环境配置
│   └── monitoring.md      # 监控方案
│
└── development/          # 开发相关文档
    ├── guidelines.md     # 开发规范
    └── testing.md        # 测试策略
```

## Documentation Guidelines

1. 所有文档使用Markdown格式编写
2. 文档应该保持简洁、清晰、准确
3. 代码示例应该包含必要的注释
4. 定期更新文档以反映最新的系统状态

## Getting Started

- 系统概述和入门指南请查看 `architecture/overview.md`
- API文档请查看 `api/endpoints.md`
- 开发指南请查看 `development/guidelines.md`
