## 背景与问题

### 核心问题

当前项目的占位符实现经常出现问题，导致翻译后的文件出现标签错误。这是一个**结构完整性**问题——EPUB文件的核心是HTML/XML结构，如果结构被破坏，翻译后的文件将无法正常打开或显示。

### 问题表现

1. **占位符被截断**：Chunk边界切断了标签对
2. **占位符被删除**：LLM在翻译时错误地删除了占位符
3. **占位符被调换**：LLM调换了占位符的顺序
4. **标签对不匹配**：开始标签和结束标签被分到不同Chunk
5. **验证失效**：当前验证只检查数量，不检查位置

```python
class Placeholder:
    characters = string.ascii_uppercase + string.digits  # 36种字符
    ID_LENGTH = 4

    def generate(self):
        # 生成随机4位字符串，如 "ABCD", "1234"
        _placeholder = "".join(secrets.choice(self.characters) for _ in range(ID_LENGTH))
        return _placeholder
```

## 参考系统分析

### 参考系统核心设计理念

1. **结构完整性优先**：宁可翻译失败，也不破坏EPUB结构
2. **多层防御机制**：三级回退策略保证鲁棒性
3. **智能分块**：基于HTML结构的智能分块，避免破坏标签
4. **占位符保护**：用占位符替换所有HTML标签，让LLM只翻译纯文本

### 参考系统占位符设计

#### 三层索引架构

```
LEVEL 1 - 文档级 (TagPreserver)
输入HTML: "<body><p>Hello</p></body>"
→ 替换为占位符: "[id0]Hello[id1]"
→ global_tag_map: {"[id0]": "<body><p>", "[id1]": "</p></body>"}

LEVEL 2 - Chunk级 (HtmlChunker)
全局文本: "[id5]Hello[id6] [id7]World[id8]"
→ Chunk 1: "[id0]Hello[id1]" (本地重新编号)
  ├─ global_indices: [5, 6]
  └─ 本地占位符: [id0], [id1]

→ Chunk 2: "[id0]World[id1]" (本地重新编号)
  ├─ global_indices: [7, 8]
  └─ 本地占位符: [id0], [id1]

LEVEL 3 - 翻译后 (PlaceholderManager)
Chunk翻译后: "[id0]Bonjour[id1]" (本地索引)
→ 恢复全局索引: "[id5]Bonjour[id6]"
```

```python
# 参考系统的标签占位符化
def preserve_tags(self, text: str) -> Tuple[str, Dict[str, str]]:
    """
    将HTML标签替换为占位符

    核心思想：相邻的标签和空白合并为一个占位符
    """
    # 按标签和非标签分割文本
    segments = re.split(r'(<[^>]+>)', text)

    for segment in segments:
        if is_tag or is_non_trans:
            # 标签或非翻译内容：加入当前组
            current_group.append(segment)
        else:
            # 可翻译文本：先flush当前组为占位符
            if current_group:
                merged_content = ''.join(current_group)
                placeholder = self.placeholder_format.create(self.counter)
                self.tag_map[placeholder] = merged_content
                merged_segments.append(placeholder)
                self.counter += 1
                current_group = []

            # 直接添加可翻译文本
            merged_segments.append(segment)

    # 处理末尾的标签组
    if current_group:
        merged_content = ''.join(current_group)
        placeholder = self.placeholder_format.create(self.counter)
        self.tag_map[placeholder] = merged_content
        merged_segments.append(placeholder)
```

### 参考系统三层防御机制

```
Phase 1: 正常翻译 + 占位符验证
    ↓ 失败
Phase 2: Token Alignment（不使用占位符翻译，然后对齐重插）
    ↓ 失败
Phase 3: 返回未翻译原文（保证结构完整）
```

```python
### 三层防御触发流程

```
翻译请求
    ↓
═══════════════════════════════
Phase 1: 正常翻译（最多max_retries次，每次失败都重试）
═══════════════════════════════
    ↓ 发送给LLM
    ↓ 验证占位符完整性
    ├─ ✅ 验证通过 → 恢复全局索引 → 返回结果
    └─ ❌ 验证失败 → 重试（直到所有重试次数用完）
                         ↓
             所有重试失败 → 进入Phase 2（只尝试一次）
                         ↓
═══════════════════════════════
Phase 2: Token Alignment Fallback
═══════════════════════════════
    ↓ 提取纯净文本
    ↓ 翻译纯净文本
    ↓ 词级对齐重插占位符
    ├─ ✅ 对齐成功 → 恢复全局索引 → 返回结果
    └─ ❌ 对齐失败 → 进入Phase 3
                         ↓
═══════════════════════════════
Phase 3: Untranslated Fallback
═══════════════════════════════
    ↓ 返回原文（保留结构完整性）
    ↓ 标记为UNTRANSLATED状态
    ↓ 返回success=True（不算失败，只是未翻译）


**注意**：
- Phase 1 失败后会重试（最多max_retries次）
- Phase 2 只在 Phase 1 所有重试都失败后尝试一次
- Phase 3 返回原文，但 status 为 UNTRANSLATED

#### Phase 2: Token Alignment Fallback

**核心思想**：既然LLM容易破坏占位符，那就不让LLM看到占位符！

```python
# 1. 提取纯净文本
clean_text = fmt.remove_all(chunk_text)
# [id0]Hello[id1] World[id2] → "Hello World"

# 2. 翻译纯净文本（不带占位符指令）

    clean_text,
    has_placeholders=False  # 不带占位符指令

# "Hello World" → "Hola Mundo"

# 3. 词级别对齐，重新插入占位符

    original_with_placeholders=chunk_text,
    translated_without_placeholders=translated_clean,
    placeholders=placeholders_list
```

**对齐算法**：

```python
def align_and_insert_placeholders(
    self,
    original_with_placeholders: str,
    translated_without_placeholders: str,
    placeholders: List[str]
) -> str:
    """通过词级别对齐重新插入占位符"""

    # 1. 找到占位符在原文中的位置
    positions = self._find_placeholder_positions(original_with_placeholders, placeholders)
    # 例如：[(5, 10, "[id0]"), (20, 25, "[id1]")]

    # 2. 计算相对位置（百分比）
    relative_positions = self._calculate_relative_positions(positions, len(original_with_placeholders))
    # 例如：[(0.1, "[id0]"), (0.4, "[id1]")]
    # [id0]在10%位置，[id1]在40%位置

    # 3. 映射到译文
    mapped_positions = self._map_to_translated(relative_positions, translated_without_placeholders)

    # 4. 调整到最近的词边界
    adjusted_positions = [
        (self._find_nearest_word_boundary(translated_without_placeholders, pos), ph)
        for pos, ph in mapped_positions
    ]

    # 5. 插入占位符
    result = self._insert_placeholders_at_positions(
        translated_without_placeholders,
        adjusted_positions,
        placeholders
    )

    return result
```

### 参考系统分块策略

```python
def chunk_html_with_placeholders(self, text_with_placeholders: str, tag_map: Dict[str, str]) -> List[Dict]:
    """将带占位符的文本分块为合适的chunk"""

    # 找到安全分割点（完整HTML块之间）
    split_points = self._find_safe_split_points(text_with_placeholders, tag_map)

    # 在分割点分割文本
    segments = self._split_at_points(text_with_placeholders, split_points)

    # 合并segment为chunk（考虑token限制）
    chunks = self._merge_segments_into_chunks(segments, tag_map)

    return chunks
```

**安全分割点优先级**：

1. **最高优先级**：章节标题后（`</h1>`, `</h2>`, `</h3>`）
2. **普通优先级**：段落、div等块级元素后（`</p>`, `</div>`）
3. **不分割**：内联标签之间（`</span>`, `</em>`等）

## 两种设计对比

### 索引方式对比

| 方面 | epubox (当前) | 参考系统 |
|------|---------------|---------|
| **索引格式** | 随机字符串 `##ABCD##` | 顺序数字 `[id0]`, `[id1]` |
| **索引层级** | 单层（全局） | 三层（全局→局部→恢复） |
| **ID生成** | `secrets.choice()` 随机 | 顺序递增 `0, 1, 2...` |
| **Chunk边界** | 可能切断占位符 | 保留标签对完整性 |
| **Token对齐** | 不支持 | 支持 |

### 标签合并对比

| 方面 | epubox (当前) | 参考系统 |
|------|---------------|---------|
| **合并策略** | 不合并，每个标签单独替换 | 相邻标签+空白合并为一个占位符 |
| **LLM干扰** | 每个标签都是独立占位符 | 合并后占位符更少，更难破坏 |

### 验证能力对比

| 方面 | epubox (当前) | 参考系统 |
|------|---------------|---------|
| **数量验证** | ✅ 有 | ✅ 有 |
| **位置验证** | ❌ 无 | ✅ 有 |
| **顺序验证** | ❌ 无 | ✅ 有 |

### 决策结论

由于当前项目的占位符实现**经常性出现问题**，导致翻译后的文件出现标签问题，我们决定采用参考系统的设计方案。具体原因：

1. **Token对齐Fallback是核心需求**
   - 当前系统的所有保护都是"预防性"的（让LLM别破坏）
   - 参考系统有"修复性"保护（破坏后能恢复）

2. **顺序数字索引更优**
   - 紧凑：`[id0]` vs `##ABCD##`
   - 可映射：知道位置关系，可以做比例映射
   - 对LLM更友好：有规律

3. **标签合并减少破坏面**
   - 合并后占位符更少
   - LLM需要保留的占位符更少

#### PlaceholderManager 类

```python
class PlaceholderManager:
    """
    管理占位符的创建、索引转换和恢复

    三层索引架构：
    1. 文档级全局索引：[id0], [id1], [id2]... (在整个文档中唯一)
    2. Chunk级局部索引：[id0], [id1]... (在每个chunk中从0开始)
    3. 全局索引恢复：将局部索引转换回全局索引
    """

    def __init__(self):
        self.tag_map: Dict[str, str] = {}  # 全局占位符→原始标签
        self.counter: int = 0               # 全局计数器

    def create_placeholder(self, tag_content: str) -> str:
        """创建一个新的占位符"""
        placeholder = f"[id{self.counter}]"
        self.tag_map[placeholder] = tag_content
        self.counter += 1
        return placeholder

    def get_local_tag_map(self, global_indices: List[int]) -> Dict[str, str]:
        """
        获取指定全局索引对应的局部tag_map

        用于chunk翻译时，告诉LLM这个chunk使用了哪些占位符
        """
        local_map = {}
        for local_idx, global_idx in enumerate(global_indices):
            placeholder = f"[id{global_idx}]"
            if placeholder in self.tag_map:
                local_map[f"[id{local_idx}]"] = self.tag_map[placeholder]
        return local_map

    def restore_to_global(
        self,
        translated_with_local: str,
        global_indices: List[int]
    ) -> str:
        """
        将翻译后的文本（使用局部索引）转换回全局索引

        支持多种占位符格式检测：
        - [id0], [id1] (标准格式)
        - [[0]], [[1]] (双括号格式)
        - {{0}}, {{1}} (双花括号格式)
        """
        result = translated_with_local

        # 1. 检测占位符格式
        if '[id' in result:
            prefix, suffix = '[id', ']'
        elif '[[' in result:
            prefix, suffix = '[[', ']]'
        elif '{{' in result:
            prefix, suffix = '{{', '}}'
        else:
            # 未知格式，使用默认值
            prefix, suffix = '[id', ']'
            logger.warning(f"未知占位符格式，尝试使用默认格式: {prefix}")

        # 2. 用临时标记替换局部占位符，避免冲突
        for local_idx in range(len(global_indices)):
            local_ph = f"{prefix}{local_idx}{suffix}"
            if local_ph in result:
                result = result.replace(local_ph, f"__RESTORE_{local_idx}__")

        # 3. 用全局索引替换临时标记
        for local_idx, global_idx in enumerate(global_indices):
            result = result.replace(f"__RESTORE_{local_idx}__", f"[id{global_idx}]")

        return result

    def remove_all_placeholders(self, text: str) -> str:
        """移除所有占位符，返回纯净文本"""
        result = text
        for placeholder in self.tag_map:
            result = result.replace(placeholder, "")
        return result
```

```python
class TagPreserver:
    """
    将HTML标签替换为占位符

    合并策略：相邻的标签和空白合并为一个占位符
    """

    BLOCK_TAGS = {
        "p", "div", "h1", "h2", "h3", "h4", "h5", "h6",
        "ul", "ol", "li", "table", "tr", "td", "th",
        "blockquote", "pre", "code", "section", "article",
        "header", "footer", "nav", "aside", "main", "figure"
    }

    INLINE_TAGS = {
        "span", "a", "em", "strong", "b", "i", "u", "s",
        "sub", "sup", "small", "mark", "cite", "q", "abbr"
    }

    IGNORE_TAGS = {
        "script", "style", "svg", "math", "img", "meta",
        "link", "br", "hr", "input", "textarea", "select"
    }

    IGNORE_CLASSES = {
        ("code", "processedcode"),
        ("div", "no-translate"),
        ("span", "notranslate"),
        ("code", "language-*"),
    }

    def __init__(self):
        self.placeholder_mgr = PlaceholderManager()

    def preserve_tags(self, html: str) -> Tuple[str, PlaceholderManager]:
        """
        将HTML标签替换为占位符

        Returns:
            (处理后的文本, PlaceholderManager实例)
        """
        result_parts = []
        segments = re.split(r'(<[^>]+>)', html)

        current_tag_group = []

        for segment in segments:
            if not segment:
                continue

            is_tag = segment.startswith('<') and segment.endswith('>')
            is_non_trans = self._is_non_translatable(segment)

            if is_tag or is_non_trans:
                # 标签或非翻译内容：加入当前组
                current_tag_group.append(segment)
            else:
                # 可翻译文本：先flush当前标签组为占位符
                if current_tag_group:
                    merged_content = ''.join(current_tag_group)
                    placeholder = self.placeholder_mgr.create_placeholder(merged_content)
                    result_parts.append(placeholder)
                    current_tag_group = []

                # 添加可翻译文本
                result_parts.append(segment)

        # 处理末尾的标签组
        if current_tag_group:
            merged_content = ''.join(current_tag_group)
            placeholder = self.placeholder_mgr.create_placeholder(merged_content)
            result_parts.append(placeholder)

        return ''.join(result_parts), self.placeholder_mgr

    def _is_non_translatable(self, tag: str) -> bool:
        """检查标签是否不可翻译"""
        # 检查是否是自闭合标签
        if re.match(r'<(meta|link|img|br|hr|input)\b', tag, re.I):
            return True

        # 检查是否是script/style等
        tag_name = re.match(r'<(\w+)', tag)
        if tag_name and tag_name.group(1).lower() in self.IGNORE_TAGS:
            return True

        return False

    def restore_tags(
        self,
        translated_text: str,
        placeholder_mgr: PlaceholderManager
    ) -> str:
        """将占位符恢复为原始标签"""
        result = translated_text

        # 按索引排序恢复（从大到小，避免替换冲突）
        sorted_items = sorted(
            placeholder_mgr.tag_map.items(),
            key=lambda x: int(x[0][3:-1]),
            reverse=True
        )

        for placeholder, original in sorted_items:
            result = result.replace(placeholder, original)

        return result
```

```python
async def translate_step(step_input: StepInput) -> StepOutput:
    chunk: Chunk = step_input.input
    placeholder_mgr = step_input.additional_data.get("placeholder_mgr")

    # 检查是否有可翻译内容
    if not _has_translatable_content(chunk.original):
        chunk.translated = chunk.original
        chunk.status = TranslationStatus.TRANSLATED
        return StepOutput(content=chunk)

    # 获取Chunk的局部tag_map和全局索引
    local_tag_map = placeholder_mgr.get_local_tag_map(chunk.global_indices)

    # Phase 1: 正常翻译（最多重试max_retries次）
    phase1_success = False
    for attempt in range(MAX_TRANSLATION_RETRIES):
        translated = await call_translator(
            chunk.original,
            local_tag_map=local_tag_map
        )

        # 验证占位符
        is_valid, corrected = _validate_and_fix_placeholders(
            chunk.original,
            translated,
            local_tag_map
        )

        if is_valid:
            # 恢复全局索引
            chunk.translated = placeholder_mgr.restore_to_global(
                corrected,
                chunk.global_indices
            )
            chunk.status = TranslationStatus.TRANSLATED
            return StepOutput(content=chunk)

        logger.warning(f"Phase 1 重试 {attempt + 1}/{MAX_TRANSLATION_RETRIES} 失败")

    # Phase 1 所有重试都失败 → 尝试 Phase 2: Token Alignment（只尝试一次）
    logger.warning("Phase 1 所有重试失败，尝试 Phase 2...")
    aligned = await token_alignment_fallback(
        chunk.original,
        local_tag_map,
        chunk.global_indices
    )

    if aligned:
        chunk.translated = aligned
        chunk.status = TranslationStatus.TRANSLATED
        return StepOutput(content=chunk)

    # Phase 2 也失败 → 进入 Phase 3: 返回原文（保留结构完整性）
    original_with_global = placeholder_mgr.restore_to_global(
        chunk.original,
        chunk.global_indices
    )
    chunk.translated = original_with_global
    chunk.status = TranslationStatus.UNTRANSLATED  # 新状态：未翻译但保留结构
    logger.error(f"Chunk '{chunk.name}': Phase 1 和 Phase 2 都失败，保留原文")
    return StepOutput(content=chunk, success=True)  # 算成功，只是未翻译
```

```python
# engine/agents/aligner.py

async def token_alignment_fallback(
    original: str,
    local_tag_map: Dict[str, str],
    global_indices: List[int]
) -> Optional[str]:
    """
    Phase 2: Token对齐Fallback

    被 translate_step 调用，作为 Phase 1 失败后的备选方案。

    流程：
    1. 提取纯净文本（移除所有占位符）
    2. 翻译纯净文本
    3. 词级别对齐，重新插入占位符
    4. 验证占位符完整性
    """

    # 1. 提取纯净文本
    clean_text = remove_placeholders(original, local_tag_map)
    # "[id0]Hello [id1]World[id2]" → "Hello World"

    if not clean_text.strip():
        return None

    # 2. 翻译纯净文本
    translated_clean = await translate_clean_text(clean_text)
    # "Hello World" → "Bonjour Monde"

    # 3. 词级别对齐
    aligned = align_and_insert_placeholders(
        original_with_placeholders=original,
        translated_without_placeholders=translated_clean,
        local_tag_map=local_tag_map
    )

    # 4. 验证
    is_valid, _ = _validate_and_fix_placeholders(original, aligned, local_tag_map)
    if not is_valid:
        logger.error("Phase 2 对齐后验证失败")
        return None

    # 5. 恢复全局索引
    result = PlaceholderManager.restore_to_global(aligned, global_indices)

    return result


def align_and_insert_placeholders(
    original_with_placeholders: str,
    translated_without_placeholders: str,
    local_tag_map: Dict[str, str]
) -> str:
    """
    通过词级别对齐重新插入占位符
    """

    # 1. 找到占位符在原文中的字符位置
    positions = []
    for placeholder in sorted(local_tag_map.keys(),
                              key=lambda x: int(x[3:-1])):
        pattern = re.escape(placeholder)
        for m in re.finditer(pattern, original_with_placeholders):
            positions.append((m.start(), m.end(), placeholder))

    if not positions:
        return translated_without_placeholders

    # 2. 计算相对位置（百分比）
    original_len = len(original_with_placeholders)
    relative_positions = []
    for start, end, placeholder in positions:
        relative_pos = start / original_len
        relative_positions.append((relative_pos, placeholder))

    # 3. 映射到译文
    translated_len = len(translated_without_placeholders)
    mapped_positions = []
    for relative_pos, placeholder in relative_positions:
        mapped_char_pos = int(relative_pos * translated_len)
        mapped_positions.append((mapped_char_pos, placeholder))

    # 4. 调整到最近的词边界
    adjusted = adjust_to_word_boundaries(
        translated_without_placeholders,
        mapped_positions
    )

    # 5. 按位置从后往前插入（避免位置偏移）
    result = translated_without_placeholders
    for char_pos, placeholder in sorted(adjusted, key=lambda x: x[0], reverse=True):
        # 确保不在单词中间插入
        char_pos = find_safe_insert_position(result, char_pos)
        result = result[:char_pos] + placeholder + result[char_pos:]

    return result


def adjust_to_word_boundaries(
    text: str,
    positions: List[Tuple[int, str]]
) -> List[Tuple[int, str]]:
    """调整位置到最近的词边界"""

    # 找到所有单词边界
    word_boundaries = set([0, len(text)])
    for match in re.finditer(r'\b', text):
        word_boundaries.add(match.start())

    adjusted = []
    for char_pos, placeholder in positions:
        # 找到最接近的词边界
        closest = min(word_boundaries, key=lambda x: abs(x - char_pos))
        adjusted.append((closest, placeholder))

    return adjusted
```

```python
# engine/agents/validator.py

def _validate_and_fix_placeholders(
    original: str,
    translated: str,
    local_tag_map: Dict[str, str]
) -> tuple[bool, str]:
    """
    增强的占位符验证：检查数量和位置
    """

    # 提取占位符
    original_placeholders = _extract_placeholder_indices(original)
    translated_placeholders = _extract_placeholder_indices(translated)

    # 1. 检查数量
    if len(original_placeholders) != len(translated_placeholders):
        logger.error(f"占位符数量不匹配: 原文{len(original_placeholders)}, 译文{len(translated_placeholders)}")
        return False, translated

    # 2. 检查位置（按顺序比较）
    position_errors = []
    for i, (orig_ph, trans_ph) in enumerate(zip(original_placeholders, translated_placeholders)):
        if orig_ph != trans_ph:
            position_errors.append(
        f"位置{i}占位符不匹配: 期望{orig_ph}, 得到{trans_ph}"
            )

    if position_errors:
        logger.error("占位符位置验证失败:")
        for err in position_errors:
            logger.error(f"  - {err}")
        return False, translated

    # 3. 验证通过
    logger.info("占位符验证通过")
    return True, translated


def _extract_placeholder_indices(text: str) -> List[str]:
    """提取文本中的所有占位符索引（按出现顺序）"""
    return re.findall(r'\[id\d+\]', text)
```

## 分块策略重构

### 核心思想

**只在完整HTML块边界分割**，保证每个chunk包含成对的占位符。

### 完整块级分割原则

**关键约束**：每个chunk必须包含成对的占位符，不允许"孤儿占位符"。

```
✅ 正确：每个chunk的占位符是完整的对
Chunk 1: [id0]Hello[id1]
Chunk 2: [id2]World[id3]

❌ 错误：占位符被切断（孤儿占位符）
Chunk 1: [id0]Hello        ← [id1]在下一个chunk
Chunk 2: [id1]World[id3]  ← [id0]在上一个chunk
```

**如何保证**：
- 在找分割点时，必须检查当前占位符对应的标签和下一个占位符对应的标签
- 如果当前是结束标签(如`</p>`)，且下一个是开始标签(如`<div>`)，这才是安全分割点
- 否则，即使在token限制内，也不能分割

**分割判断逻辑**：

```python
def _is_safe_split_point(
    current_tag_content: str,  # 当前占位符代表的标签内容
    next_tag_content: str      # 下一个占位符代表的标签内容
) -> bool:
    """
    判断是否是安全分割点

    安全分割点条件：
    1. 当前是块级结束标签（如 </p>, </div>）
    2. 下一个是块级开始标签（如 <p>, <div>）
    """
    current_is_block_close = self._is_block_closing_tag(current_tag_content)
    next_is_block_open = self._is_block_opening_tag(next_tag_content)

    return current_is_block_close and next_is_block_open
```

**示例**：

```
全局文本：[id5]<h1>Chapter 1</h1>[id6]<p>Hello</p>[id7]<span>World</span>[id8]</p>[id9]

标签内容：
[id5] = "<h1>"
[id6] = "</h1>"
[id7] = "<p>"
[id8] = "<span>"
[id9] = "</span></p>"

分割点分析：
- 位置id5：当前"<h1>"是开始标签，不是结束标签 → 不分割
- 位置id6：当前"</h1>"是结束标签，下一个"<p>"是开始标签 → ✅ 安全分割点！
- 位置id7：当前"<p>"是开始标签 → 不分割
- 位置id8：当前"<span>"是开始标签 → 不分割
- 位置id9：当前"</span></p>"包含结束标签，但下一个没有内容 → 不分割

分割结果：
Chunk 1: [id0]<h1>Chapter 1</h1>[id1]    (在</h1>后分割)
Chunk 2: [id0]<p>Hello</p>[id1]<span>World</span>[id2]  (在</p>后分割)
```

### 设计

```python
# engine/item/chunker.py

class HtmlChunker:
    """
    基于HTML结构的智能分块器

    设计原则：
    1. 只在完整HTML块边界分割
    2. 分割前分析标签对完整性
    3. 确保不会切断标签对
    """

    # 允许作为分割点的块级标签
    ALLOWED_SPLIT_TAGS = {
        "p", "div", "h1", "h2", "h3", "h4", "h5", "h6",
        "li", "ul", "ol", "blockquote", "pre", "section",
        "article", "header", "footer", "nav", "aside",
        "main", "figure", "figcaption", "table", "tr"
    }

    # 禁止分割的标签（必须保持完整）
    FORBIDDEN_SPLIT_TAGS = {
        "navPoint", "navLabel", "content", "navMap",
        "pageList", "pageTarget", "spine", "itemref"
    }

    def __init__(self, token_limit: int = 3000):
        self.token_limit = token_limit
        self.placeholder_pattern = re.compile(r'\[id\d+\]')

    def chunk(
        self,
        text_with_placeholders: str,
        global_indices: List[int],
        placeholder_mgr: PlaceholderManager,
        is_nav_file: bool = False
    ) -> List[Chunk]:
        """
        将带占位符的文本分块

        Args:
            text_with_placeholders: 带占位符的HTML文本
            global_indices: 文本中所有占位符的全局索引
            placeholder_mgr: PlaceholderManager实例
            is_nav_file: 是否是导航文件

        Returns:
            List[Chunk]: 分块后的列表
        """

        # Step 1: 找到所有占位符的位置
        placeholder_positions = self._find_placeholder_positions(
            text_with_placeholders
        )

        # Step 2: 找到安全分割点
        split_points = self._find_safe_split_points(
            text_with_placeholders,
            placeholder_positions,
            placeholder_mgr,
            is_nav_file
        )

        # Step 3: 根据分割点分割文本
        segments = self._split_at_points(text_with_placeholders, split_points)

        # Step 4: 合并segments为chunk（考虑token限制）
        chunks = self._merge_into_chunks(
            segments,
            global_indices,
            placeholder_mgr
        )

        return chunks

    def _find_placeholder_positions(self, text: str) -> List[Tuple[int, int, str, int]]:
        """
        找到所有占位符的位置

        Returns:
            List of (start, end, placeholder, index)
            例如: [(0, 4, '[id0]', 0), (10, 15, '[id1]', 1), ...]
        """
        positions = []
        for match in self.placeholder_pattern.finditer(text):
            placeholder = match.group()
            index = int(placeholder[3:-1])
            positions.append((match.start(), match.end(), placeholder, index))
        return positions

    def _find_safe_split_points(
        self,
        text: str,
        placeholder_positions: List[Tuple[int, int, str, int]],
        placeholder_mgr: PlaceholderManager,
        is_nav_file: bool
    ) -> List[int]:
        """
        找到安全分割位置

        分割点优先级：
        1. 块级结束标签后（如 </p>, </div>）
        2. 章节标题后（</h1>, </h2>, </h3>）
        3. 列表项后（</li>）
        """

        split_points = []
        allowed_tags = self.ALLOWED_SPLIT_TAGS if not is_nav_file else set()

        for i, (start, end, placeholder, idx) in enumerate(placeholder_positions):
            tag_content = placeholder_mgr.tag_map.get(placeholder, "")

            # 跳过禁止分割的标签
            if is_nav_file and tag_content.startswith('<'):
                tag_name = self._extract_tag_name(tag_content)
                if tag_name in self.FORBIDDEN_SPLIT_TAGS:
                    continue

            # 检查是否是块级结束标签
            if self._is_block_closing_tag(tag_content):
                # 检查下一个是否是块级开始标签
                if i + 1 < len(placeholder_positions):
                    next_placeholder = placeholder_positions[i + 1][2]
                    next_tag = placeholder_mgr.tag_map.get(next_placeholder, "")
                    if self._is_block_opening_tag(next_tag):
                        # 找到一个安全分割点
                        priority = self._get_split_priority(tag_content)
                        split_points.append((end, priority))

        # 按优先级排序，返回位置
        split_points.sort(key=lambda x: x[1], reverse=True)  # 高优先级优先
        return [pos for pos, _ in split_points]

    def _is_block_closing_tag(self, tag: str) -> bool:
        """检查是否是块级结束标签"""
        match = re.match(r'</([a-zA-Z][a-zA-Z0-9]*)\s*>', tag)
        if match:
            tag_name = match.group(1).lower()
            return tag_name in self.ALLOWED_SPLIT_TAGS
        return False

    def _is_block_opening_tag(self, tag: str) -> bool:
        """检查是否是块级开始标签"""
        match = re.match(r'<([a-zA-Z][a-zA-Z0-9]*)\b', tag)
        if match:
            tag_name = match.group(1).lower()
            return tag_name in self.ALLOWED_SPLIT_TAGS
        return False

    def _get_split_priority(self, tag: str) -> int:
        """获取分割优先级"""
        match = re.match(r'</([a-zA-Z][a-zA-Z0-9]*)\s*>', tag)
        if match:
            tag_name = match.group(1).lower()
            if tag_name in {"h1", "h2", "h3"}:
                return 100  # 最高优先级
            if tag_name == "p":
                return 50
            if tag_name in {"li", "div"}:
                return 30
        return 10

    def _split_at_points(
        self,
        text: str,
        split_points: List[int]
    ) -> List[str]:
        """在分割点分割文本"""
        if not split_points:
            return [text]

        all_points = sorted(set([0] + split_points + [len(text)]))
        segments = []
        for i in range(len(all_points) - 1):
            segment = text[all_points[i]:all_points[i + 1]]
            if segment:
                segments.append(segment)
        return segments

    def _merge_into_chunks(
        self,
        segments: List[str],
        global_indices: List[int],
        placeholder_mgr: PlaceholderManager
    ) -> List[Chunk]:
        """将segments合并为考虑token限制的chunk"""

        chunks = []
        current_chunk_parts = []
        current_indices = []

        for segment in segments:
            # 计算如果加入这个segment，token数是否超限
            test_parts = current_chunk_parts + [segment]
            test_text = ''.join(test_parts)
            test_tokens = count_tokens(test_text)

            if test_tokens <= self.token_limit or not current_chunk_parts:
                # 可以加入
                current_chunk_parts.append(segment)
                # 收集这个segment中的占位符索引
                for ph in self.placeholder_pattern.findall(segment):
                    idx = int(ph[3:-1])
                    if idx not in current_indices:
                        current_indices.append(idx)
            else:
                # 保存当前chunk，开始新的
                if current_chunk_parts:
                    chunks.append(self._create_chunk(
                        current_chunk_parts,
                        current_indices,
                        placeholder_mgr
                    ))
                current_chunk_parts = [segment]
                current_indices = []
                # 收集这个segment中的占位符索引
                for ph in self.placeholder_pattern.findall(segment):
                    idx = int(ph[3:-1])
                    if idx not in current_indices:
                        current_indices.append(idx)

        # 保存最后一个chunk
        if current_chunk_parts:
            chunks.append(self._create_chunk(
                current_chunk_parts,
                current_indices,
                placeholder_mgr
            ))

        return chunks

    def _create_chunk(
        self,
        parts: List[str],
        indices: List[int],
        placeholder_mgr: PlaceholderManager
    ) -> Chunk:
        """创建一个Chunk对象"""
        text = ''.join(parts)
        return Chunk(
            name=str(uuid.uuid4())[:8],
            original=text,
            global_indices=sorted(indices),
            local_tag_map=placeholder_mgr.get_local_tag_map(sorted(indices)),
            tokens=count_tokens(text)
        )
```

```python
# engine/agents/validator.py

def validate_placeholder_positions(
    original: str,
    translated: str,
    local_tag_map: Dict[str, str]
) -> tuple[bool, str, List[str]]:
    """
    验证占位符位置

    Returns:
        (is_valid, corrected_text, errors)
    """

    original_indices = extract_placeholder_indices(original)
    translated_indices = extract_placeholder_indices(translated)

    errors = []

    # 1. 数量验证
    if len(original_indices) != len(translated_indices):
        errors.append(
            f"占位符数量不匹配: 原文{len(original_indices)}, 译文{len(translated_indices)}"
        )
        return False, translated, errors

    # 2. 顺序验证
    for i, (orig_idx, trans_idx) in enumerate(zip(original_indices, translated_indices)):
        if orig_idx != trans_idx:
            errors.append(
                f"位置{i}占位符索引不匹配: 原文[{orig_idx}], 译文[{trans_idx}]"
            )

    if errors:
        return False, translated, errors

    # 3. 验证通过
    return True, translated, []


def extract_placeholder_indices(text: str) -> List[int]:
    """提取占位符的索引值"""
    matches = re.findall(r'\[id(\d+)\]', text)
    return [int(m) for m in matches]
```

### HTML结构完整性验证

```python
# engine/agents/verifier.py

def verify_html_integrity(html: str) -> bool:
    """
    验证HTML标签是否正确闭合
    """

    # 使用栈来跟踪标签
    stack = []
    i = 0
    n = len(html)

    while i < n:
        if html[i] == '<':
            # 找到标签开始
            j = html.find('>', i)
            if j == -1:
                return False  # 未找到标签结束

            tag = html[i:j + 1]

            # 跳过自闭合标签
            if is_self_closing(tag):
                i = j + 1
                continue

            # 跳过注释和DOCTYPE
            if tag.startswith('<!--') or tag.startswith('<!'):
                i = j + 1
                continue

            # 检查是否是结束标签
            if tag.startswith('</'):
                tag_name = get_tag_name(tag)
                if not tag_name:
                    i = j + 1
                    continue

                if stack and stack[-1] == tag_name:
                    stack.pop()
                elif tag_name in stack:
                    # 标签交错
                    return False
                # else: 未匹配的结束标签，忽略

            else:
                # 开始标签
                tag_name = get_tag_name(tag)
                if tag_name and not is_self_closing(tag):
                    stack.append(tag_name)

            i = j + 1
        else:
            i += 1

    # 检查是否所有标签都闭合
    return len(stack) == 0


def is_self_closing(tag: str) -> bool:
    """检查是否是自闭合标签"""
    self_closing = {
        'br', 'hr', 'img', 'input', 'meta', 'link',
        'area', 'base', 'col', 'embed', 'param', 'source', 'track', 'wbr'
    }
    tag_name = get_tag_name(tag)
    if tag_name in self_closing:
        return True
    return tag.endswith('/>')


def get_tag_name(tag: str) -> Optional[str]:
    """从标签中提取标签名"""
    match = re.match(r'</?([a-zA-Z][a-zA-Z0-9]*)\b', tag)
    if match:
        return match.group(1).lower()
    return None
```

```python
class Replacer:
    def restore(self, item: EpubItem):
```
